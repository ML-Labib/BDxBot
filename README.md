BDxBot - Tournament Manager:

A custom Discord bot engineered for BD-Extreme Esports, a premier PUBG community in Bangladesh. This bot automates the workflow of hosting competitive esports tournaments, reducing manual workload for administrators.

Key Features:

Dynamic VC Management: Automatically creates and assigns private Voice Channels for participating teams.
Anti-Cheat Integration: Handles MOSS file submissions and verification for fair play.
Role Automation: Manages permissions and role assignments for players and staff.
Auto-Cleanup: Automatically purges channels and roles after tournaments to keep the server organized.
